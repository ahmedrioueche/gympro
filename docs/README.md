# GymPro Documentation

Welcome to the official documentation for **GymPro**, a comprehensive management and tracking system for gyms, coaches, and members.

- [Backend Documentation](./backend.md) – NestJS architecture, MongoDB schema, and service integrations.

## Project Structure

GymPro is organized as a monorepo:

- `/backend` – NestJS API (Node.js)
- `/web` – React (Vite) frontend + Capacitor (Mobile)
- `/packages/client` – Shared type-safe API client
- `/docs` – Project documentation (You are here)
